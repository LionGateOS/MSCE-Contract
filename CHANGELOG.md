# Changelog

## v1.0.0

Initial MSCE Memory Contract release.

Included:

- JSON Schema definition;
- contract validation tooling;
- validation tests;
- reference payload examples;
- versioning documentation.

Guarantees:

- stable contract identity;
- lifecycle state validation;
- provenance requirements;
- confidence boundaries;
- timestamp validation.

Not included:

- WorkRunner adapters;
- Memory adapters;
- execution logic;
- storage logic.

Those remain owned by their respective systems.
